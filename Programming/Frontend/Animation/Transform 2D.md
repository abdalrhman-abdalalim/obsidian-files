- allows many properties 
	- `translate()`
		- function moves an element from its current position (according to the parameters given for the X-axis and the Y-axis).
	- `rotate()`
		-  function rotates an element clockwise or counter-clockwise according to a given degree
	- `scaleX()`
		- The `scaleX()` function increases or decreases the width of an element.
	- `scaleY()`
		- The `scaleX()` function increases or decreases the height of an element.
	- `scale()`
		- function increases or decreases the size of an element (according to the parameters given for the width and height).
	- `skewX()`
		- skew means (انحراف) along the x-axis
	- `skewY()`
		-  skew means (انحراف) along the y-axis
	- `skew()`
		- function skews an element along the X and Y-axis by the given angles.
	- `matrix()`
		- function combines all the 2D transform functions into one.
		- The parameters are as follow: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY()) 

- origin (x,y) -> the origin of transform 


- how to measure properties :
	- deg -> Degree
	- rad -> Radians
	- grad -> Gradians
	- turn -> turn
