The `metadata` export is a **special object** in Next.js **used to define SEO-related metadata for a page or layout**.

### âœ… **Example of Metadata Usage**

```tsx
export const metadata: Metadata = {
  title: "Create Next App",  // Page title shown in the browser tab
  description: "Generated by create next app",  // Meta description for SEO
};
```

This will generate:

```html
<title>Create Next App</title>
<meta name="description" content="Generated by create next app">
```

Next.js **automatically applies this metadata** to the `<head>` of your HTML document.
