
### **Available `::-webkit-scrollbar` Selectors**

| Selector                          | Description                                                         |
| --------------------------------- | ------------------------------------------------------------------- |
| `::-webkit-scrollbar`             | The entire scrollbar                                                |
| `::-webkit-scrollbar-thumb`       | The draggable handle                                                |
| `::-webkit-scrollbar-track`       | The track where the thumb moves                                     |
| `::-webkit-scrollbar-track-piece` | The part of the track not covered by the thumb                      |
| `::-webkit-scrollbar-button`      | Buttons at the ends of the scrollbar                                |
| `::-webkit-scrollbar-corner`      | The bottom-right corner where vertical & horizontal scrollbars meet |
| `::-webkit-resizer`               | The draggable resizer in bottom-right corner (if resizable)         |
